Функции:
Очистка временных файлов Windows
Удаление кэша браузеров Chrome, Firefox и Edge
Программа не трогает Cookies или другие файлы кроме временных папок и файлов

Системные требования:
Операционная система: От Windows 8.1
Процессор: От 1 ГГц
Оперативная память: От 512 МБ
Свободное место на диске: От 50 МБ

Установка:
Скачайте файл Cleaner.exe
Запустите программу от имени администратора для полного доступа ко всем временным файлам
При первом запуске программа создаст точку восстановления системы
Готово можно пользоваться

Использование:
Запустите Cleaner.exe
Нажмите кнопку Очистить для поиска временных файлов
После завершения очистки можете закрыть программу
В ней нету автозагрузки или работы в фоне

Предупреждение!
Перед использованием программы рекомендуется создать резервную копию важных данных
Программа не удаляет системные файлы необходимые для работы Windows
При возникновении проблем используйте функцию восстановления системы
